①

6억원을 30년 만기로 대출받았으므로

매년 균등하게 상환해야 하는 원금은

6억원 / 30년 = 2,000만원

1회차 원리금(원금 + 이자)이 4,400만원 이고

원금이 2,000만원 이므로

2,000 + 이자 = 4,400만원

이자 = 2,400원


따라서, 금리는

이자 / 대출액 = 2,400만원 / 6억원 = 4%


회차별 이자와 원리금은


■ 1회차

●이자 = 6억원 × 4% = 2,400만원

●원리금 = 2,000만원 + 2,400만원 = 4,400만원


■ 2회차

1회차에서 원금 2,000만원을 갚았으므로

●이자 = (6억원 - 2,000만원) × 4% = 2,320만원

●원리금 = 2,000만원 + 2,320만원 = 4,320만원


■ 3회차

2회차에서 원금 2,000만원을 갚았으므로

●이자 = (5억8천만원 - 2,000만원) × 4% = 2,240만원

●원리금 = 2,000만원 + 2,240만원 = 4,240만원



원금균등분할상환을 공식으로 정리하면

(대출금/대출기간) + {대출금 - [(대출금/대출기간) × (회차 - 1)]} × 대출금리
