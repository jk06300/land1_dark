일반적으로

수요의 가격탄력성을 구할 때

아래 공식을 사용하는데

수요의 가격탄력성

=수요량변화율 / 가격변화율

=(수요량변동분 / 최초수요량) / (가격변동분 / 최초가격)


위 공식으로 계산했을 때 답이 안나오는 경우에는

중간점을 이용한 계산법을 사용한다.

중간점을 이용한 계산법에서는

변동분을 최초값과 최후값의 중간값으로 나누어 변화율을 계산한다.

즉,

수요의 가격탄력성

=수요량변화율 / 가격변화율

=[수요량변동분/ ((최초수요량 + 최후수요량) / 2)] / [가격변동분 / ((최초가격 + 최후가격) / 2)]


=[(108 - 92) / ((108 + 92) / 2)] / [(11 - 9) / ((11 + 9) / 2)]

=(16/100) / (2/10)

=0.16 / 0.2

=0.8
