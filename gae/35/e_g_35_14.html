컨버스의 분기점모형 공식


A에서 경계지점 까지의 거리

= AB 거리 / [1 + √(B 인구/A 인구)]


= 45 / [1 + √(21/84)]

= 45 / (1 + 0.5) = 30㎞



※ 다른 풀이방법 (1)

A와 B의 인구 비

A : B = 84 : 21 = 4 : 1 이므로


경계지점에서

(A까지 거리)² : (B까지 거리)² = 4 : 1

제곱을 없애면

(A까지 거리) : (B까지 거리) = 2 : 1


따라서, 경계지점에서 A까지의 거리는

45㎞ 중 2/(2+1) 이므로

45㎞ × 2/3 = 30㎞



※ 다른 풀이방법 (2)

A로부터 경계지점 까지의 거리를 X 라고 가정하면

A 인구 / X² = B 인구 / (30㎞ - X)²

(45㎞ - X)² / X² = B 인구 / A 인구

(45 - X)² / X² = 21 / 84

[ (45 - X) / X ]² = 1/4

제곱을 없애면

(45 - X) / X = 1/2

45 - X = 1/2 × X

45 = 1/2X + X

45 = 3/2X

45 × 2/3 = X

X = 30㎞
