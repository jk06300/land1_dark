⑤


　 가능총소득　　2,000만원

－ 공실손실상당액  500만원

　 유효조소득　　1,500만원

－ 영업소득　　　1,000만원

　 순영업소득　　  500만원

－ 부채서비스액　  500만원

　 세전현금수지　　　  0원



①

담보인정비율 (LTV)

= 주택담보대출액 / 부동산가치

= 1억원 / 2억원 = 0.5



②

부채감당률 (DCR)

= 순영업소득 / 부채서비스액

= 500만원 / 500만원 = 1.0



③

총부채상환비율 (DTI)

= 부채서비스액 / 연소득

= 500만원 / 1,250만원 = 0.4



④

채무불이행률 (DR)

= (영업경비 + 부채서비스액) / 유효조소득

= (1,000만원 + 500만원) / 1,500만원

= 1.0



⑤

영업경비비율 (유효총소득 기준)

= 영업경비 / 유효총소득

= 1,000 / 1,500 = 0.67
