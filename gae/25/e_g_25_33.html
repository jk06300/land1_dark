④

D사업에만 투자하는 경우는 C와 D사업에 투자하는 경우보다 수익성지수가 더 작다.


아래 풀이에서와 같이


D사업에만 투자하는 경우

수익성 지수 = 1.07


C와 D사업에 투자하는 경우

수익성지수 = 1.08



◆ 문제풀이


수익성지수

= 수익현가 / 초기의 현금지출

수익현가 ( r = 할인율, n = 기간)

= 말기의 현금유입 × 1 / (1 + r)^n


■ A

수익현가

= 121 × 1/(1 + 0.1)¹) = 110

수익성지수

= 110 / 100 = 1.1


■ B

수익현가

= 130 × 1/(1 + 0.1)¹) = 118.2

수익성지수

= 118.2 / 120 = 0.98


■ C

수익현가

= 180 × 1/(1 + 0.1)¹) = 163.6

수익성지수

= 163.6 / 150 = 1.09


■ D

수익현가

= 200 × 1/(1 + 0.1)¹) = 181.8

수익성지수

= 181.8 / 170 = 1.07


■ C + D

수익현가

= 380 × 1/(1 + 0.1)¹) = 345.5

수익성지수

= 345.5 / 320 = 1.08
