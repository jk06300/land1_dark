④

지분환원율

=세전현금수지 / 지분투자금액

=(순영업소득 - 부채서비스액) / 지분투자금액


= (8천만원 - 4천만원) / 5억원

= 4천만원 / 5억원 = 0.08 = 8%



①

종합환원율

= 순영업소득 / 총투자금액

= 8천만원 / 8억원 = 0.1 = 10%


②

저당비율

= 대부액 / 총투자금액


= 3억원 / 8억원 = 0.375 = 37.5%


③

자본회수기간

= 총투자금액 / 순영업소득


= 8억원 / 8천만원 = 10년


⑤

부채감당율

= 순영업소득 / 부채서비스액

= 8천만원 / 4천만원 = 2


※ 다른 풀이 방법

●지문마다 지표의 값이 주어지는 문제에서는

●혹시 역수관계가 있는지 살펴본다.


역수관계인 지표끼리는

둘을 곱할 경우 100%(= 1)가 되므로

있을 경우 둘을 곱해

●100%(= 1)가 되면 둘 다 정답이 아니고

●안되면 둘 중 하나가 정답이다.


이 문제에서는

●①번과 ③번이 역수관계인데 (위 공식 참조)

●둘을 곱하면 100%가 되므로

●이 둘은 정답이 될 수 없다.


따라서, 정답은 ②, ④, ⑤ 중에 있을 것이므로

●②, ④, ⑤ 중 자신있는 지표부터 풀어본다면

●문제 푸는 시간을 좀 더 단축할 수 있을 것이다.
