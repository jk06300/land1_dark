상권분석모형 공식


A의 시장점유율  (n = 마찰계수)

＝A의 고객유인력

　／(A + B + C)의 고객유인력

＝[ A면적 / (AX거리)ⁿ ]

　／[ A면적 / (AX거리)ⁿ ＋ B면적 / (BX거리)ⁿ ＋ C면적 / (CX거리)ⁿ ]


■증축 전

＝(500/5²)

　／[ (500/5²) ＋ (2000/10²) ＋ (1000/5²) ]

＝20 / (20 + 20 + 40)

＝20 / 80 = 0.25  (25%)


■증축 후

＝(2500/5²)

　／[ (2500/5²) ＋ (2000/10²) ＋ (1000/5²) ]

＝20 / (100 + 20 + 40)

＝100 / 160 = 0.625  (62.5%)


●증축 전 점유율 25%일 때 월매출액 4억원

●증축 후 점유율 62.5%일 때 월매출액 X


따라서

25 : 4 = 62.5 : X

25 * X = 4 * 62.5

25X = 250

X = 10(억원)
