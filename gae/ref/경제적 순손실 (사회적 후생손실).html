경제적 순손실 (사회적 후생손실)

정부의 세금부과 등과 같은 시장 왜곡현상에 따라 초래되는 총잉여(총이익)의 감소분을 말한다.

경제적 순손실

=총잉여의 감소분

=세금부과전 총잉여 - 세금부과후 총잉여


예를 들어

●볼펜의 생산원가가 90원이고

●판매가격은 100원

●소비자가 볼펜에 지불할 용의가 있는 금액은 110원일 경우


생산자는

●생산원가가 90원인 볼펜을

●100원에 판매했으므로


생산자잉여

= 100원 - 90원 = 10원

소비자는

●110원까지 지불할 의사가 있는데

●100원에 구입했으므로


소비자잉여

= 110원 - 100원 = 10원

총잉여

= 생산자잉여 + 소비자잉여

= 10원 + 10원 = 20원


만약 볼펜 거래량이 10개일 경우

●생산자잉여 = 10원 × 10개 = 100원

●소비자잉여 = 10원 × 10개 = 100원

●총잉여 = 100원 + 100원 = 200원


그런데 정부가

●볼펜에 세금 20원을 부과해서

●생산자가 판매가격을 110원으로 인상하게 됐고

●그 결과 거래량이 5개로 감소한 경우

,

생산자잉여

= (판매가 - 생산원가 - 세금) × 거래량

= (110원 - 90원 - 20원) × 5개 = 0원


소비자잉여

= (지불의사금액 - 판매가) × 거래량

= (110원 - 110원) × 5개 = 0원


정부잉여

= 세금 × 거래량

= 20원 × 5개 = 100원


총잉여

= 생산자잉여 + 소비자잉여 + 정부잉여

= 0원 + 0원 + 100원 = 100원


총잉여가

●세금부과 전 200원에서

●세금부과 후 100원이 되면서

●100원이 감소하게 되는데


이와 같은 총잉여의 감소분을

경제적 순손실 또는 사회적 후생손실이라고 한다.
