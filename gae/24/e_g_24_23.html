①

순현가

= 수익현가 - 초기의 현금지출

수익현가 ( r = 할인율, n = 기간)

= 말기의 현금유입 × 1/(1 + r)^n


투자안 A의

수익현가

= 6,000 × 1/(1 + 0.2)¹)

= 5,000

순현가

= 5,000 - 5,000 = 0원


내부수익률은

순현가 = 0 또는

수익성지수가 1 이 되는

할인율이므로


순현가 = 수익현가 - 초기의 현금지출

계산식에서 순현가를 0 으로 놓으면


0 = 말기현금유입 × 1/(1 + r)ⁿ

　　- 초기현금지출


0 = 6,000원 × 1/(1 + r)ⁿ - 5,000원

5,000원 = 6,000원 × 1/(1 + r)

5,000원 / 6,000원 = 1/(1 + r)

6,000원 / 5,000원 = 1 + r

1.2 = 1 + r

r = 1.2 - 1 = 0.2


따라서,

내부수익률 = 20%
