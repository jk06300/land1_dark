다음과 같은 수요곡선이 있을 경우

●가격이 1,000원일 때 수요량 1,000개

●가격이 100원 상승할 때 수요량 100개 감소

●가격이 100원 하락할 때 수요량 100개 증가


이 수요곡선의 어떤 점에서 측정하든

기울기 = 수요량 / 가격 = 1이 된다.

그런데, 수요의 가격탄력성은

●수요량 / 가격이 아니라

●수요량변화율 / 가격변화율 이므로


측정하는 점의 위치에 따라

값이 달라지게 된다.


예를 들어, 가격이 1,000원에서 200원 오르면

수요량은 1,000개에서 200개 감소하므로

●가격변화율 = 200 / 1,000 = 0.2

●수요량변화율 = 200 / 1,000 = 0.2

●가격탄력성 = 0.2 / 0.2 = 1


오른 가격 1,200원에서 다시 600원이 더 오르면

감소한 수요량 800개에서 다시 600개가 감소하므로

●가격변화율 = 600 / 1,200 = 0.5

●수요량변화율 = 600 / 800 = 0.75

●가격탄력성 = 0.75 / 0.5 = 1.5


따라서, 우하향하는 선분으로 주어진 수요곡선의 경우

수요곡선상의 측정지점에 따라 가격탄력성은 다르게 나타난다.
