③

입지계수 (LQ)

＝특정산업의 지역고용률

　／특정산업의 전국고용률

＝(지역의 특정산업 고용자수／

　 지역의 전체산업 고용자수)

　／(전국의 특정산업 고용자수／

　　 전국의 전체산업 고용자수)


따라서,

각 도시별 X 산업의 입지계수는


■ A 도시

= (400 / 1000) / (3350 / 6250)

= (400 * 6250) / (1000 * 3350)

= 0.75


■ B 도시

= (1200 / 2000) / (3350 / 6250)

= (1200 * 6250) / (2000 * 3350)

≒ 1.12


■ C 도시

= (650 / 1150) / (3350 / 6250)

= (650 * 6250) / (1150 * 3350)

≒ 1.05


■ D 도시

= (1100 / 2100) / (3350 / 6250)

= (1100 * 6250) / (2100 * 3350)

≒ 0.98
