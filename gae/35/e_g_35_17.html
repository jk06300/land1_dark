현재가치

= 미래의 예상가격 / (1 + r)ⁿ


●r = 할인율 (요구수익률)

●n = 기간


■1년 후 지하철 역사가 개발될 경우 A토지 예상가격의 현재가치

= 14억 3천만원 / (1 + 0.1)^1

= 14억 3천만원 / 1.1

= 13억원 ---- ㉠


■1년 후 지하철 역사가 개발되지 않을 경우 A토지 예상가격의 현재가치

= 8억 8천만원 / (1 + 0.1)^1

= 8억 8천만원 / 1.1

= 8억원 ---- ㉡


정보의 현재가치

= (㉠ - ㉡) × 안될 확률

= (13억원 - 8억원) × 40%

= 2억원



※ 간편 계산식

정보의 현재가치 = [ (성공 예상값－실패 예상값) × 실패확률 ] / (1＋할인율)ⁿ

따라서,

[ (14억 3천만원 - 8억 8천만원) × 0.4 ] / (1 + 0.1)^1

= 2억원
