컨버스의 분기점모형 공식


A에서 경계선 까지의 거리

= AB 거리 / [1 + √(B 면적/A 면적)]


= 8 / [1 + √(9000/1000)]

= 8 / (1 + 3)

= 2㎞



※ 다른 풀이방법 (1)

A와 B의 면적 비

A : B = 1000 : 9000 = 1 : 9 이므로


경계선에서

(A까지 거리)² : (B까지 거리)² = 1 : 9

제곱을 없애면

(A까지 거리) : (B까지 거리) = 1 : 3


따라서, 경계선에서 A까지의 거리는

8㎞ 중 1/(1+3) 이므로

8㎞ × 1/4 = 2㎞



※ 다른 풀이방법 (2)

A로부터 경계선 까지의 거리를 X 라고 가정하면

A 면적 / X² = B 면적 / (8㎞ - X)²

(8㎞ - X)² / X² = B 면적 / A 면적

(8 - X)² / X² = 9000 / 1000

[ (8 - X) / X ]² = 9

제곱을 없애면

(8 - X) / X = 3

8 - X = 3 × X

8 = 3X + X

8 = 4X

X = 2㎞
