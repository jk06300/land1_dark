④

지분환원율

= 지분투자자 몫 / 지분투자금액


지분투자자 몫 (세전현금수지)

= 순영업소득 - 부채서비스액


= 8천만원 - 4천만원

= 4천만원


따라서,

지분환원율

= 4천만원 / 5억원 = 8%



①

종합환원율

= 순영업소득 / 총투자금액


총투자금액

= 지분투자금액 + 저당액


=5억원 + 3억원 = 8억원


따라서,

종합환원율

= 8천만원 / 8억원 = 10%



②

저당비율

= 저당액 / 총투자금액


= 3억원 / 8억원 = 37.5%



③

자본회수기간

= 총투자금액 / 순영업소득


= 8억원 / 8천만원 = 10년



⑤

부채감당율

= 순영업소득 / 부채서비스액


= 8천만원 / 4천만원 = 2
