④

(종합)자본환원율

= 순영업소득 / 총투자액


순영업소득

= 세전현금수지 + 부채서비스액


= 6,000만원 + 4,000만원

= 1억원


따라서,

(종합)자본환원율

= 1억원 / 10억원 = 10%



①

(유효)총소득

= 총투자액 / 총소득승수


= 10억원 / 5 = 2억원



②

순소득승수

= 총투자액 / 순영업소득


순영업소득

= 세전현금수지 + 부채서비스액


= 6,000만원 + 4,000만원

= 1억원


따라서,

순소득승수

= 10억원 / 1억원 = 10



③

세전현금수지승수

= 지분투자액 / 세전현금수지


= 6억원 / 6,000만원 = 10



⑤

부채감당율

= 순영업소득 / 부채서비스액


순영업소득

= 세전현금수지 + 부채서비스액


= 6,000만원 + 4,000만원

= 1억원


따라서,

부채감당율

= 1억원 / 4,000만원

= 2.5
