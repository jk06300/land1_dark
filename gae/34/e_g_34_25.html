③

지분환원율 (= 지분배당률)

세전현금흐름 / 지분투자액

= (순영업소득 - 원리금상환액) / 지분투자액

= (2억원 - 2천만원) / 8억원 = 22.5%



①

유효총소득

총투자액 / 유효총소득승수

= 10억원 / 4 = 2억5천만원



②

부채비율

부채 / 지분투자액

= (총투자액 - 지분투자액) / 지분투자액

= (10억원 - 8억원) / 8억원

= 2억원 / 8억원 = 25%



④

순소득승수

총투자액 / 순영업소득

= 10억원 / 2억원 = 5

종합환원율 (순소득승수와 역수관계)

순영업소득 / 총투자액

= 2억원 / 10억원 = 0.2
