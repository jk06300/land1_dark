⑤

현재가치

= 미래의 예상가격 / (1 + r)ⁿ


●r = 할인율 (요구수익률)

●n = 기간


■2년 후 복합쇼핑몰이 개발될 경우 토지의 예상가격의 현재가치

= 6억 500만원 / (1 + 0.1)^2

= 6억 500만원 / 1.21

= 5억원 ---- ㉠


■2년 후 복합쇼핑몰이 개발되지 않을 경우 토지의 예상가격의 현재가치

= 3억 250만원 / (1 + 0.1)^2

= 3억 250만원 / 1.21

= 2억 5천만원 ---- ㉡


정보의 현재가치

= (㉠ - ㉡) × 안될 확률

= (5억원 - 2억 5천만원) × 50%

= 2억 5천만원 × 50%

= 1억 2천 5백만원



※ 간편 계산식

정보의 현재가치 = [ (성공 예상값－실패 예상값) × 실패확률 ] / (1＋할인율)ⁿ

따라서,

[ (6억 500만원 - 3억 250만원) × 0.5 ] / (1 + 0.1)^2

= 1억 2천 5백만원
