①

●대출액이 6억원

●대출기간이 30년이고

●원금균등상환방식이므로


매년 균등하게 상환해야 하는 원금은

6억원 / 30년 = 2,000만원

1회차 원리금 상환액이 4,400만원이라고 했으므로

4,400만원 = 원금균등액 2,000만원 + 1회차 이자

1회차 이자 = 4,400만원 - 2,000만원 = 2,400만원

1회차 이자 = 2,400만원


대출금리

= 이자 / 대출액

= 2,400만원 / 6억원 = 0.04 = 4%



3회차에 상환할 원리금은

다음과 같이 계산한다.

n회차 원리금

= 원금균등액 + n회차 이자

n회차 이자

=1회차 이자 - 매기간 이자감소액 × (n - 1)

1회차에는 이자가 감소하지 않으므로 (n - 1)을 곱한다.

매기간 이자감소액은

= 1회차 이자 / 대출기간

= 2,400만원 / 30년 = 80만원 이므로


3회차 이자는

= 2,400만원 - 80만원 × (3 - 1)

= 2,400만원 - 160만원 = 2,240만원


따라서, 3회차 원리금은

2,000만원 + 2,240만원 = 4,240만원


※ 다른 풀이 방법

●3회차 원리금을 구할 때

●위 방법이 생각나지 않을 경우

●아래 방법으로 구할 수도 있다.


■ 1회차

●이자 = 6억원 × 4% = 2,400만원

●원리금 = 2,000만원 + 2,400만원 = 4,400만원


■ 2회차

1회차에서 원금 2,000만원을 갚았으므로

●이자 = (6억원 - 2,000만원) × 4% = 2,320만원

●원리금 = 2,000만원 + 2,320만원 = 4,320만원


■ 3회차

2회차에서 원금 2,000만원을 갚았으므로

●이자 = (5억8천만원 - 2,000만원) × 4% = 2,240만원

●원리금 = 2,000만원 + 2,240만원 = 4,240만원
