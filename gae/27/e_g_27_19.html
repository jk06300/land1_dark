③  ㉡, ㉣



㉠

타인자본의 이용으로 레버리지를 활용하면

부채비율 상승으로 인해 금융위험이 높아진다.


㉡

대부비율

= 부채총액 / 총자본

= 부채총액 / (자기자본 + 부채총액)


대부비율이 50% 라고 했으므로

●총자본을 100 으로 봤을 때

●부채총액 50, 자기자본 50 이 된다.


자기자본수익률

＝종합수익률 + (종합수익률 - 저당수익률) × 부채투자 / 지분투자

= 0.1 + (0.1 - 0.08) × 50/50

= 0.1 + 0.02 × 1 = 0.12

= 12%



㉢

예를 들어

●부채 1억원

●이자율(저당수익률) 5% (= 년 5백만원)

●자기자본 1억원

●순영업소득 5백만원일 경우


총자본수익률

= 순영업소득 / (자기자본 + 부채)

= 5백만원 / (1억원 + 1억원)

= 2.5%


총자본수익률이 저당수익률보다 작으므로

부(-)의 레버리지가 된다.


이 경우

●부채를 5천만원으로 줄이고

●자기자본을 1억5천만원으로 늘리면?


5백만원 / (5천만원 + 1억5천만원)

= 5백만원 / 2억원 = 2.5%


여전히

총자본수익률은 2.5%로 변하지 않는다.

즉, 이자율보다 낮은 수익을 내는 상황에서는

●부채비율을 낮춘다고 해서

●정(+)의 레버리지로 전환되는 것은 아니며

●근본적으로 수익(순영업소득)을 많이 내서

●총자본수익률을 이자율보다 높여야 한다.
