# 거미집 유형별 특성


■ A 주택시장

Qd = 200 - P

P = -Qd + 200

수요기울기 절대값 = [-1] = 1

Qs = 100 + 4P

4P = Qs - 100

P = (Qs - 100) / 4 = ¼Qs - 25

공급기울기 절대값 = [1/4] = 0.25


공급기울기 절대값이

수요기울기 절대값보다 작으므로 발산형


■ B 주택시장

Qd = 500 - 2P

2P = -Qd + 500

P = (-Qd + 500) / 2 = -0.5Qd + 250

수요기울기 절대값 = [-0.5] = 0.5

Qs = 200 + ½P

½P = Qs - 200

P = (Qs - 200) × 2 = 2Qs - 400

공급기울기 절대값 = [2] = 2


공급기울기 절대값이

수요기울기 절대값보다 크므로 수렴형
