상권분석모형 공식

A의 시장점유율  (n = 마찰계수)

＝A의 고객유인력

　／(A + B + C)의 고객유인력

＝[ A면적 / (AX거리)ⁿ ]

　／[ A면적 / (AX거리)ⁿ ＋ B면적 / (BX거리)ⁿ ＋ C면적 / (CX거리)ⁿ ]


＝(500/5²)

　／[ (500/5²) ＋ (300/10²) ＋ (450/15²) ]

＝20 / (20 + 3 + 2)

＝20 / 25 = 0.8  (80%)


따라서,

●할인점 A의 시장점유율은 80%

●X 지역 주민은 4,000명

●1인당 월 할인점 소비액은 35만원이고

●주민 모두가 구매자이므로


4,000명 × 80% × 35만원

＝11억 2,000만원



※ 다른 풀이방법

매장 유인력 = 면적 / 거리²

●A의 유인력 = 500/5² = 20

●B의 유인력 = 300/10² = 3

●C의 유인력 = 450/15² = 2


A의 시장점유율

= 20 / (20 + 3 + 2) = 0.8  (80%)

●할인점 A의 시장점유율은 80%

●X 지역 주민은 4,000명

●1인당 월 할인점 소비액은 35만원이고

●주민 모두가 구매자이므로


할인점 A의 이용고객수는

4,000명 × 80% × 35만원

＝11억 2,000만원
