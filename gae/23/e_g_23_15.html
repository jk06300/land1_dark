⑤

A의 순현재가치는 4,000만원,

D는 3,000만원이므로

4,000 / 3,000 ≒ 1.3


◆ 문제풀이


순현가

= 수익현가 - 초기의 현금지출

수익성지수

= 수익현가 / 초기의 현금지출

수익현가 ( r = 할인율, n = 기간)

= 말기의 현금유입 × 1 / (1 + r)^n


■ A

수익현가

= 7,490 × 1 / (1 + 0.07)¹ = 7,000

순현가

= 7,000 - 3,000 = 4,000

수익성지수

= 7,000 / 3,000 = 2.3


■ B

수익현가

= 2,675 × 1 / (1 + 0.07)¹ = 2,500

순현가

= 2,500 - 1,000 = 1,500

수익성지수

= 2,500 / 1,000 = 2.5


■ C

수익현가

= 3,210 × 1 / (1 + 0.07)¹ = 3,000

순현가

= 3,000 - 1,500 = 1,500

수익성지수

= 3,000 / 1,500 = 2


■ D

수익현가

= 4,815 × 1 / (1 + 0.07)¹ = 4,500

순현가

= 4,500 - 1,500 = 3,000

수익성지수

= 4,500 / 1,500 = 3
