④

PI가 가장 큰 순서는 C > B > D > A 이다.


순현가

= 수익현가 - 금년의 현금지출

수익성지수

= 수익현가 / 금년의 현금지출

수익현가 ( r = 할인율, n = 기간)

= 내년의 현금유입 × 1/(1 + r)ⁿ


■ A

수익현가

= 630 × 1/(1 + 0.05)¹) = 600

순현가

= 600 - 300 = 300

수익성지수

= 600 / 300 = 2


■ B

수익현가

= 315 × 1/(1 + 0.05)¹) = 300

순현가

= 300 - 100 = 200

수익성지수

= 300 / 100 = 3


■ C

수익현가

= 420 × 1/(1 + 0.05)¹) = 400

순현가

= 400 - 100 = 300

수익성지수

= 400 / 100 = 4


■ D

수익현가

= 262.5 × 1/(1 + 0.05)¹) = 250

순현가

= 250 - 100 = 150

수익성지수

= 250 / 100 = 2.5


따라서,


각 투자안의 순현가 크기 순서는

A = C > B > D


수익성지수 크기 순서는

C > B > D > A



⑤

총투자비용이 300만원일 경우


A 투자안만 투자할 경우

순현가 = 300만원


B, C, D 투자안에 투자할 경우

순현가

= 200만원 + 300만원 + 150만원

= 650만원


따라서,

A 투자안만 투자하는 것보다

B, C, D 투자안에 투자하는 것이 유리하다.
