④

(종합)자본환원율

=순영업소득 / 총투자액

=(세전현금수지 + 부채서비스액) / 총투자액


= (6,000만원 + 4,000만원) / 10억원

= 1억원 / 10억원

= 0.1 = 10%



①

(유효)총소득승수

●유효총소득으로 총투자액을 회수하는데 걸리는 시간

●총투자액 / (유효)총소득


5 = 10억원 / (유효)총소득

(유효)총소득 = 10억원 / 5 = 2억원



②

순소득승수

●순영업소득으로 총투자액을 회수하는데 걸리는 시간

●총투자액 / 순영업소득= 총투자액 / (세전현금수지 + 부채서비스액)


= 10억원 / (6,000만원 + 4,000만원)

= 10억원 / 1억원 = 10



③

세전현금수지승수

●세전현금수지로 지분투자액을 회수하는데 걸리는 시간

●지분투자액 / 세전현금수지


= 6억원 / 6,000만원 = 10


⑤

부채감당율

=순영업소득 / 부채서비스액

=(세전현금수지 + 부채서비스액) / 부채서비스액


= (6,000만원 + 4,000만원) / 4,000만원

= 1억원 / 4,000만원= 2.5



※ 다른 풀이 방법

●②번 순소득승수와 ④번 자본환원율은 역수관계이므로 (위 공식 참조)

●② × ④ = 100% (= 1) 가 되어야 한다.


그런데 ②와 ④를 곱하면 80%가 나오므로

둘 중 하나는 틀렸다는 것을 알 수 있다.

따라서, ②번과 ④번 중 하나를 계산해 보면

둘 중 어떤게 틀렸는지 알 수 있을 것이다.
