부채감당률

1.5 = 1,000만원 / χ

①

지분배당률

= 세전현금수지 / 지분투자액


= 300만원 / 6000만원 = 0.05

= 5%



②

부채비율

= 부채총액 / 자기자본


= 4억원 / 8억원 = 0.5

= 50%



③

자본환원율

= 순영업소득 / 총투자액


= 2,100만원 / 3억원 = 0.07

= 7%



④

자기자본비율

= 자기자본 / 총투자액


= 3억원 / 4억원 = 0.75

= 75%
