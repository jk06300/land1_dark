수요의 교차탄력성

=A의 수요량변화율 / B의 가격변화율

=(A의 수요량변화분 / A의 원래수요량) / (B의 가격변화분 / B의 원래가격)


수요의 교차탄력성이 0.8이라고 가정했으므로

0.8 = [ (X - 1200) / 1,200 ] / [ (2,000 - 1600) / 1,600 ]

0.8 = [ (X - 1200) / 1,200 ] / (400 / 1,600)

0.8 = [ (X - 1200) / 1,200 ] / 0.25

(X - 1,200) / 1,200 = 0.8 × 0.25

X - 1,200 = 0.2 × 1,200

X = 240 + 1,200 = 1,440
