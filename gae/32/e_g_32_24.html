⑤

수익현가 ( r = 할인율, n = 기간)

= 말기의 현금유입 × 1 / (1 + r)^n

순현가

= 수익현가 - 초기의 현금지출

수익성지수

= 수익현가 / 초기의 현금지출


■ A

수익현가

= 6,825 × 1 / (1 + 0.05)¹ = 6,500

순현가

= 6,500 - 3,800 = 2,700

수익성지수

= 6,500 / 3,800 = 1.71


■ B

수익현가

= 2,940 × 1 / (1 + 0.05)¹ = 2,800

순현가

= 2,800 - 1,250 = 1,550

수익성지수

= 2,800 / 1,250 = 2.24

수익현가

= 4,725 × 1 / (1 + 0.05)¹ = 4,500

수익성지수
