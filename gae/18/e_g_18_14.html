③

입지계수가 상대적으로 크면

상대적으로 특화된 산업이다.

따라서, 각 산업별로

●A 도시와 B 도시의 입지계수를 구해

●A 도시의 입지계수가 B 도시보다 큰 산업을 찾는다.



■ 제조업

A

= (300/1000) / (4000/10000)

= 0.3/0.4 = 0.75

B

= (1200/3000) / (4000/10000)

= 0.4/0.4 = 1


■ 금융업

A

= (500/1000) / (5000/10000)

= 0.5/0.5 = 1

B

= (1500/3000) / (5000/10000)

= 0.5/0.5 = 1


■ 부동산업

A

= (200/1000) / (1000/10000)

= 0.2/0.1 = 2  (B 보다 큼)

B

= (300/3000) / (1000/10000)

= 0.1/0.1 = 1



※ 다른 풀이 방법

●위에 계산식을 살펴보면

●A와 B 둘 다 분모가 같으므로


두 도시의 분자 크기만 비교해도

A 도시가 B 도시에 비해 특화되어 있는 산업이 어떤 건지 찾을 수 있다.

■ 제조업

A 300 / 1000 = 0.3
B 1200 / 3000 = 0.4

■ 금융업

A 500 / 1000 = 0.5
B 1500 / 3000 = 0.5

■ 부동산업

A 200 / 1000 = 0.2  (B 보다 큼)
B 300 / 3000 = 0.1
