시장상황이

불황일 때 수익률 10%, 확률 30%

보통일 때 수익률 20%, 확률 40%

호황일 때 수익률 30%, 확률 30%이므로


전체 기대수익률

= (0.1×0.3) + (0.2×0.4) + (0.3×0.3)

= 0.03 + 0.08 + 0.09

= 0.2 = 20%


분산은 각각의 시장상황별 수익률에서

평균수익률을 뺀 값의 제곱에

확률을 곱한 값들을 합하면 된다.


평균수익률

= (10% + 20% + 30%) / 3

= 60% / 3 = 20%


분산

= (0.1 - 0.2)² × 0.3

+ (0.2 - 0.2)² × 0.4

+ (0.3 - 0.2)² × 0.3


= (-0.1)² × 0.3

+ (0)² × 0.4

+ (0.1)² × 0.3


= 0.01 × 0.3

+ 0 × 0.4

+ 0.01 × 0.3


= 0.003 + 0.003 = 0.006

= 0.6%
