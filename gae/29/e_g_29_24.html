③

자기자본수익률

= 총자본수익 / 자기자본

자기자본

㉠타인자본을 활용하지 않은 경우 10억원
㉡타인자본을 50% 활용하는 경우 = 10억원 × 50% = 5억원

총자본수익

= 순영업소득 + 자본이득 - 이자비용

순영업소득 = 3천만원
자본이득 = 총투자액 × 가격상승률 = 10억원 × 2% = 2천만원

이자비용

㉠타인자본을 활용하지 않은 경우 0원

㉡타인자본을 50% 활용하는 경우 = 10억원 × 50% × 이자율 = 5억원 × 4% = 2천만원


㉠타인자본을 활용하지 않은 경우

총자본수익 = 3천만원 + 2천만원 = 5천만원
자기자본수익률 = 5천만원 / 10억원 = 5%

㉡타인자본을 50% 활용하는 경우

총자본수익 = 3천만원 + 2천만원 - 2천만원 = 3천만원
자기자본수익률 = 3천만원 / 5억원 = 6%
